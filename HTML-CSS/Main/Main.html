HTML

HTML (HyperText Markup Language) is the standard language for creating web pages.

  • HTML is used to structure content on the web.
  • Elements include headings, paragraphs, links, images, and more.
  • It works alongside CSS and JavaScript.

CSS

CSS (Cascading Style Sheets) is used for styling web pages.

  • CSS controls layout, colors, and fonts.
  • It allows for responsive design using media queries.
  • CSS frameworks like Bootstrap help in quick designing.

JavaScript

JavaScript is a powerful scripting language for web development.

  • JS makes web pages interactive.
  • It can manipulate the DOM dynamically.
  • JavaScript is used in frontend & backend (Node.js).
